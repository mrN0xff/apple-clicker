apple clicker (или apple SDVG clicker) это linux x64 программа на чистом nasm ассемблере
проект создавался по фану, как породия cookie clicker

для запуска введите 
chmod +x apple_clicker
./apple_clicker

если linux дистрибутив выдаёт ошибку перекомпелируйте программу встроенным скриптом
1. установите nasm (sudo apt install nasm на ubuntu/debian)
2. введи
   
chmod +x cp

./cp

3.запускай

если у тебя windows 10 нажми win+x, выбери powershell и пропиши wsl --install: пк перезагрузится и при запуске появится консоль, это мини linux внутри винды.
